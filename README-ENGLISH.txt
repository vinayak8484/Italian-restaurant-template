──────────────────────────────────────────────────────────────
  GETTING STARTED
──────────────────────────────────────────────────────────────

  1. Open the HTML file in your browser — it works
     immediately, no installation needed.

  2. To edit it, open the file in any text editor
     (Notepad, TextEdit, VS Code, etc.)

  3. Use CTRL+F (or CMD+F on Mac) to find the text
     you want to change.

  4. Save the file and refresh the browser to see
     your changes.


──────────────────────────────────────────────────────────────
  THE 5 THINGS TO CHANGE FIRST
──────────────────────────────────────────────────────────────

  1. RESTAURANT NAME
     Search for the placeholder name (e.g. "Fiore Caffè",
     "Oro", "FUOCO") and replace it with yours.
     The name appears both in the HTML and in the JS
     translations object (search for `translations`).

  2. ADDRESS & CONTACT DETAILS
     Search for the placeholder address (e.g. "Via dei Fiori")
     and insert your real address.
     Do the same for the phone number and email.

  3. MENU & PRICES
     The menu lives inside the `translations` object in
     the JavaScript. Search for "menu" and update the dishes
     and prices for both IT and EN versions.

  4. OPENING HOURS
     Search for "Lun – Ven" (or similar) and update
     with your actual opening hours.

  5. GOOGLE MAPS
     Go to maps.google.com → search your address →
     click "Share" → "Embed a map" → copy the iframe code.
     In the HTML file, find the `<iframe` tag in the map
     section and replace the entire tag with yours.


──────────────────────────────────────────────────────────────
  LANGUAGE SWITCHER (IT / EN)
──────────────────────────────────────────────────────────────

  All templates have a language selector in the navbar
  with two pill buttons: IT | EN

  • Default language is ITALIAN.
  • Users can switch to English with one click.
  • The choice is saved in the browser.

  To add or edit translations, find the object
  `const translations = { it: {...}, en: {...} }` in the
  JavaScript and update the values for both languages.


──────────────────────────────────────────────────────────────
  DARK MODE
──────────────────────────────────────────────────────────────

  Every template has a dark/light mode button in the navbar
  (sun/moon icon). The user's preference is saved
  automatically in the browser.

  Fiore Caffè and FUOCO default to light mode.
  Oro Fine Dining defaults to dark mode (it suits the style).


──────────────────────────────────────────────────────────────
  COLOURS
──────────────────────────────────────────────────────────────

  The main colours are defined as CSS variables at the top
  of the file, inside the `:root { ... }` block.

  To change the primary colour, find the relevant variable
  and enter your colour as a HEX value.

  Example:
    --terracotta: #C17F5A;   →  swap for your colour
    --sage: #8A9E76;          →  e.g. #4A90D9 for a blue

  PHOTOS
──────────────────────────────────────────────────────────────

  The templates use placeholder photos. To use your own:

  OPTION A (recommended) — Upload to Cloudinary (free):
    1. Create an account at cloudinary.com
    2. Upload your photos
    3. Copy the URL and paste it where you see an
       "unsplash.com" or placeholder link in the HTML.

  OPTION B — Local photos:
    Put your photos in the same folder as the HTML file,
    then use: <img src="photo.jpg" ...>
