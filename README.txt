──────────────────────────────────────────────────────────────
  LE 5 COSE DA CAMBIARE SUBITO
──────────────────────────────────────────────────────────────

  1. NOME DEL RISTORANTE
     Cerca il nome del ristorante nel file (es. "Fiore Caffè",
     "Oro", "FUOCO") e sostituiscilo con il tuo.
     Il nome appare sia nell'HTML che nelle traduzioni JS
     (cerca l'oggetto `translations` nel codice).

  2. INDIRIZZO E CONTATTI
     Cerca "Via dei Fiori" (o l'indirizzo di esempio) e
     inserisci il tuo indirizzo reale.
     Cerca il numero di telefono e sostituiscilo con il tuo.
     Fai lo stesso per l'email.

  3. MENU E PREZZI
     Il menu è nella sezione `translations` nel JavaScript.
     Cerca "menu" e aggiorna piatti e prezzi per IT e EN.

  4. ORARI DI APERTURA
     Cerca "Lun – Ven" (o simile) e aggiorna gli orari
     con quelli reali del tuo locale.

  5. MAPPA GOOGLE
     Vai su maps.google.com → cerca il tuo indirizzo →
     clicca "Condividi" → "Incorpora una mappa" → copia
     il codice iframe.
     Nel file HTML cerca `<iframe` nella sezione della mappa
     e sostituisci l'intero tag con il tuo.


──────────────────────────────────────────────────────────────
  CAMBIO LINGUA (IT / EN)
──────────────────────────────────────────────────────────────

  Tutti i template hanno un selettore lingua nella navbar
  con due pulsanti: IT | EN

  • Di default la lingua è ITALIANO.
  • L'utente può passare all'inglese con un click.
  • La scelta viene salvata nel browser.

  Per aggiungere o modificare traduzioni, trova l'oggetto
  `const translations = { it: {...}, en: {...} }` nel codice
  JavaScript e aggiorna i valori per entrambe le lingue.


──────────────────────────────────────────────────────────────
  DARK MODE
──────────────────────────────────────────────────────────────

  Ogni template ha un pulsante dark/light mode nella navbar
  (icona sole/luna). La preferenza dell'utente viene salvata
  automaticamente nel browser.

  Fiore Caffè e FUOCO partono in modalità chiara.
  Oro Fine Dining parte in modalità scura (è nel suo stile).


──────────────────────────────────────────────────────────────
  COLORI
──────────────────────────────────────────────────────────────

  I colori principali sono definiti come variabili CSS in
  cima al file, nella sezione `:root { ... }`.

  Per cambiare il colore principale, trova la variabile
  corrispondente e inserisci il tuo colore in formato HEX.

  Esempio:
    --terracotta: #C17F5A;   →  cambia con il tuo colore
    --sage: #8A9E76;          →  es. #4A90D9 per un blu

     Go to maps.google.com → search your address →
     click "Share" → "Embed a map" → copy the iframe code.
     In the HTML file, find the `<iframe` tag in the map
     section and replace the entire tag with yours.
